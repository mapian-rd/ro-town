# Ro Town

This project was formerly intended to RO (Ragnarok Online) 's damage calculation, and now it is still the same.

## Feature

It's not finished yet. But I hope it has all the following functions.
1. Equipment crafting and storage
2. Status summary
3. Dps calculator
4. Custom crafting
5. Import / Export data
6. Pretty summary

### P.S.

This project is based on Thailand's ROGGT version. Therefore, some may be different, such as aspd.

- React
